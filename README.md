An AI-powered RAG (Retrieval-Augmented Generation) chatbot that allows you to upload PDFs, index them using FAISS, and chat with them using Groq's Llama-3 models.

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Configure AWS: `aws configure`
4. Run the app: `streamlit run app.py`
